# rivers-kanban-board

## Description 

This project will allow you to use a Kanban board to map out tasks into three different categories. Kanban boards are often used within the devloper community and workforce, but a Kanban board can be useful for a wide variety of things; from simple household chores to long term goals.

## Installation

GitHub Link: https://github.com/RiverStephenson/rivers-kanban-board


## Usage

Login and create tickets specifying certain tasks and place them into one of three categories; TO-DO, In-Progress and Completed. When looking at the Kanban board you can transfer tasks from category to category, edit tasks or just delete them entirely. You can also choose to logout or create a new ticket.
